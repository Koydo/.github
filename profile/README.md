# Koydo

Koydo builds AI-native lifelong learning products for families, exam prep, multilingual learning, and open curriculum infrastructure.

We care about practical learning systems that work across languages, devices, and real-world constraints. Our public work is focused on clear learning principles, trustworthy documentation, and responsible paths for future community collaboration.

## What Is Public Here

Koydo's public GitHub presence currently contains general information, public principles, and non-code documentation. Product repositories, learner data systems, infrastructure, and unreleased app code remain private while launch, safety, security, and review work continue.

## How We Work

- Learner usefulness comes first.
- Public-facing language should be clear, family-safe, and reviewer-safe.
- AI should support learning where it improves clarity, practice, accessibility, or feedback.
- Curriculum infrastructure should be explainable, portable, and respectful of local context.
- Privacy, safety, and accessibility are product requirements, not finishing touches.
